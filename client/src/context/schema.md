## Schema Explanation: `schema.js`

This file defines two Mongoose schemas: `categorySchema` and `recipeSchema`. These schemas model the data for recipe categories and individual recipes, respectively. The data for these schemas is intended to be populated from the JSON files located in the `src/Data/` directory.

### `categorySchema`

This schema defines the structure for recipe categories.

-   **`name`**:
    -   **Type**: `String`
    -   **Description**: The name of the category. It is a required, unique, and trimmed field.
    -   **Source**: The `enum` values (`'Chicken'`, `'Dessert'`, `'Seafood'`, `'Vegetarian'`, `'General'`) directly correspond to the names of the JSON files in the `src/Data/` directory:
        -   `'Chicken'`: `chickenRecipes.json`
        -   `'Dessert'`: `dessert-recipes.json`
        -   `'Seafood'`: `seafood-recipes.json`
        -   `'Vegetarian'`: `vegetarian-recipes.json`
        -   `'General'`: `recipe-data.json`

### `recipeSchema`

This schema defines the structure for individual recipes. The data for these fields is sourced from the JSON objects within the files in the `src/Data/` directory.

-   **`title`**:
    -   **Type**: `String`
    -   **Description**: The title of the recipe.
    -   **Source File**: `title` field from a recipe object in any of the `.json` files in `src/Data/`.

-   **`image`**:
    -   **Type**: `String`
    -   **Description**: The URL or path to the recipe's image.
    -   **Source File**: `image` field from a recipe object in any of the `.json` files in `src/Data/`.

-   **`description`**:
    -   **Type**: `String`
    -   **Description**: A brief description of the recipe.
    -   **Source File**: `description` field from a recipe object in any of the `.json` files in `src/Data/`.

-   **`ingredients`**:
    -   **Type**: `[String]`
    -   **Description**: A list of ingredients for the recipe.
    -   **Source File**: `ingredients` field from a recipe object in any of the `.json` files in `src/Data/`.

-   **`steps`**:
    -   **Type**: `[String]`
    -   **Description**: The steps to prepare the recipe.
    -   **Source File**: `steps` field from a recipe object in any of the `.json` files in `src/Data/`.

-   **`category`**:
    -   **Type**: `mongoose.Schema.Types.ObjectId`
    -   **Description**: A reference to the `Category` model, linking the recipe to its respective category.
    -   **Source**: This field will be populated with the `_id` of the corresponding category document.

-   **`createdAt`**:
    -   **Type**: `Date`
    -   **Description**: The date when the recipe was created. Defaults to the current date and time.
    -   **Source**: This field is automatically generated by MongoDB.
